# **App Name**: ChanloPay

## Core Features:

- Verified Host Login: Secure login system with email authentication exclusively for verified hosts.
- KYC Verification: Admin-verified host KYC process before event creation, ensuring fraud prevention.
- Dynamic QR Code Generation: Create dynamic QR codes for each event to accept digital Chanlo payments.
- Payment Tracking: Real-time transaction verification, categorization, and synchronization, even in low-network conditions.
- Reporting and Analytics Dashboard: Detailed reports including payment methods and breakdowns with visualizations that highlight transaction patterns to reveal payment trends for different types of gifts.
- Shareable Receipt Generation: QR codes that automatically track successful transactions, allowing guests to get detailed shareable receipts.
- Reasoning tool to identify potentially fraudulent transactions: AI Tool assesses transaction data, patterns, and anomalies using fuzzy logic. Flags transactions requiring manual review, enhancing fraud prevention and reducing risk of misuse.

## Style Guidelines:

- Primary color: Deep violet (#9400D3), for a regal and sophisticated feel that reflects the significance of marriage events.
- Background color: Light violet (#E6E0EB), for a calm and inviting backdrop that doesn't distract from the content.
- Accent color: Crimson (#D30028) to add contrast, signal user actions, and reinforce the app’s purpose of processing payments.
- Body text: 'PT Sans' sans-serif font for a clean, readable interface.
- Headline font: 'Playfair', a modern serif for titles. Note: currently only Google Fonts are supported.
- Clean, modern icons for easy navigation and understanding of features.
- Intuitive, user-friendly interface with clear sections for event management, payment tracking, and host verification.
- Subtle animations to indicate transaction confirmations and loading states.
